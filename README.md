## interView Task

Interview Task is a web application based on a laravel framework.
The program is a cart of products, App can be evaluated that cart by country, Offers are available on that cart.

### Available Models
- User
- Category
- Product
- Cart
  - CartItems
- Offer
  - discount
    - discountType


install App :

``` 
npm install

php artisan migrate --seed

npm run dev
```
